# Reglas del proyecto portalcbi

## Listados de estudiantes

**Todos los listados de estudiantes, sin excepción, deben comenzar con el código del estudiante (`CODIGO`) como primera columna.**

Esto aplica a cualquier tabla, reporte, exportación o vista que muestre filas de estudiantes, incluyendo planillas de notas, asistencia, listados especiales, reportes, boletines, etc.
